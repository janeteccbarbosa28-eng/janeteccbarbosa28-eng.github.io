# Janete Barbosa — Portfolio Website

Personal portfolio website

**Live site:** https://janeteccbarbosa28-eng.github.io

---

## About

Single-page portfolio built with HTML, CSS, and JavaScript. Hosted for free on GitHub Pages.

Features:
- Hero section with photo and availability status
- About Me with key stats
- Core Tech Stack organised by category
- 6 end-to-end ML projects with GitHub links
- Education section (IronHack Bootcamp + Bachelor's in Management)
- CV request via Google Form
- AI chatbot powered by LLaMA 3.3 via Groq API

---

| Layer | Technology |
